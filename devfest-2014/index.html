İşi büyütelim


angular.module('demo', [])
    .directive('modal', function () {
    return {
      transclude: true,
      template: '<div ng-show="visible">....<h1>{{ title }}</h1><section class="modal-content"></section>'+
        '<div><button ng-click="close()">Kapat</button></div>',
      scope: {
        title: '@',
        visible: '='
      },
      controller: function ($scope, $element, $attrs, $transclude) {
        $scope.close = function () {
          $scope.visible = false;
        }

        $transclude(function (clone) {
          $element.find('section').append(clone);
        });
      }
    }
});
                    

<div ng-app="demo">
  <button ng-click="modalVisible=true">Pencere ac</button>

  <modal data-title="Pencere Basligi" visible="modalVisible">
    <p>Pencere icerigi</p>
  </modal>
</div>

CSS ile hükmetmek de mümkün
(daha da güzel)

AngularJS verilen doğrulama kurallarına göre form etiketine ve form alanlarına aşağıdaki class'ları ekler.

  • ng-valid form ya da field kurallara uygun
  • ng-invalid form ya da field kurallara uygun değil
  • ng-pristine form ya da field hiç değiştirilmemiş
  • ng-dirty form ya da field değiştirilmiş
  • ng-submitted form submit edilmiş

* :valid, :invalid pseudo seçiciler de kullanılabilir.

CSS ile form doğrulama gösterimi


.form input.ng-dirty.ng-invalid {
    border: solid 1px red;
}

.form input.ng-dirty.ng-valid {
    border: solid 1px green;
}

Filtre yazmak çocuk oyuncağı

Filtreler bir veriyi sunmadan önce manipülasyonlar yapmaya yarar.


angular.module('safe', [])
    .filter('truncate', function () {
        'use strict';

        return function (text, length, end) {
            if (text) {
                length = length || 10;
                end = end || '...';

                if (text.length <= length || text.length - end.length <= length) {
                    return text;
                } else {
                    return String(text).substring(0, length - end.length) + end;
                }
            }
        };
    });
                    

<input type="text" ng-model="message" placeholder="Tweet mesaji yazin">

<p>Mesajiniz: <br>{{ message | truncate:137:'...' }}</p>
